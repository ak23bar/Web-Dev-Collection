About CourseBot

CourseBot is an AI teaching assistant that helps instructors provide personalized support at scale. By integrating with your existing learning management system, CourseBot can:

  • Answer common student questions using course materials
  • Provide detailed explanations with source references
  • Help debug code and explain programming concepts
  • Escalate complex questions to human instructors
  • Learn from past interactions to improve responses

CourseBot helps instructors save time while ensuring students get prompt, accurate assistance whenever they need it.

Example Interactions

Student:

When is Project 2 due? And can I submit it late?

CourseBot:

Project 2 is due on October 15th at 11:59 PM. According to the late policy in the syllabus, you can submit up to 48 hours late with a 10% penalty per day.

Source: Course Syllabus, Section 3.2: Late Policy
Student:

For the final project, can we use external libraries or do we need to implement everything from scratch?

CourseBot:

For the final project, you may use the external libraries listed in the project spec (NumPy, Pandas, and Matplotlib). Any other libraries require instructor approval via Piazza at least 7 days before the deadline.

Source: Final Project Guidelines, Page 2, "External Libraries"
Student:

I'm getting a NullPointerException in the LinkedList implementation. How do I fix this?

CourseBot:

This issue was discussed in a previous thread. The most common cause is not checking if the head is null before accessing it in the remove() method.

Related Thread: "Help with LinkedList implementation"

TA Response: "Make sure to add a null check before accessing head.next in your remove() method. The code should look like:

if (head != null) { // access head.next here }
Related Thread: Piazza Post #42
